# To-Do List App

เว็บแอปจัดการงานที่ต้องทำ (To-Do List)  
สามารถเพิ่ม ลบ แก้ไข และติ๊กว่างานเสร็จแล้วได้  
ข้อมูลเก็บไว้ใน LocalStorage ของเบราว์เซอร์  

---

# ฟีเจอร์
- เพิ่มงานใหม่  
- ลบงาน  
- แก้ไขชื่องาน  
- ทำเครื่องหมายว่างานเสร็จแล้ว  
- บันทึกข้อมูลใน LocalStorage  
- รองรับ responsive

---

# เทคโนโลยีที่ใช้
- Frontend: HTML, CSS, JavaScript  
  (Next.js + TailwindCSS)  
- เก็บข้อมูล LocalStorage
